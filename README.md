# Kavya Kanaja

Kavya Kanaja is an AI-powered Kannada poetry learning and exploration Android application developed using Kotlin and Jetpack Compose. The application combines traditional Kannada literature with modern Artificial Intelligence features to create an interactive learning experience for users.

The app helps users read, understand, analyze, and interact with Kannada poems through AI-based explanations, quizzes, transliteration, simplification, narration, and poem generation.

# How to Run the Project

## Prerequisites

* Android Studio
* Kotlin Support
* Android SDK
* Internet Connection

## Steps

1. Clone the repository

```bash
git clone <repository-link>
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Connect an Android device or start an emulator

5. Run the application
